```html 自然辩证法 [各门科学的辩证内容]

z = F(x, y)；间接前提是 x 变成 x + h，则

F(x + h, y) = (泰勒展开) z + dzdx h……；再让 y 变成 y + k，则

F(x + h, y + k) = z + dzdy k + dzdx h……

现在令 h = dx, k = dy，因而 k = dydx h。

因此 F(x + h, y + k) = z + (dzdy · dydx + dzdx) h……

F(x + h, y + k) - z (或 F(x, y)) = 上式右边去掉 z。

两边除以 h，或者 dx，我们得到

Dzdx = dzdy · dydx + dzdx

z' = dzdx + dzdy y'

z'' = Ddx (dzdx) + Ddy (dzdy y')

= d2zdx2 + y' d2zdx dy + y' d2zdy2 dydx + y' d2zdx dy + y'' dzdy

= d2zdx2 + 2y' d2zdx dy + y'2 d2zdy2 + y'' dzdx
