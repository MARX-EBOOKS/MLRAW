Renten verhalten sich wie 0 : 1 : 2 : 3 : 4 auch dann, wenn sich die Frucht-
barkeitsgrade verhalten wie n : n + 1 : n + 2 : n + 3 : n + 4; die Renten
verhalten sich nicht wie die Fruchtbarkeitsgrade, sondern wie die Frucht-
barkeitsunterschiede, von dem rentelosen Boden als dem Nullpunkt an
gerechnet. |
