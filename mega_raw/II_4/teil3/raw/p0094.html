Wir haben:

r · v c + v   = (r + δ)v x + v  ;                       Oder r · v c + v   = r′v x + v

∴ r c + v   = (r + δ) x + v ;                         r c + v   = r′ x + v  .

r(x + v)  = (r + δ)(c + v)              r(x + v) =r′(c + v)

    x + v = (r + δ)(c + v) r                x + v = r′ r (c + v)

       x = (r + δ)(v + c) r – v.                x = r′ r (c + v) – v.

                                                       x = r′ r (C) –v.

Da r = 100 100   = 1 1   = 1, so

x = (r + δ)(c + v) − v.

x = (r + δ)C − v

= 11 10 (500) − 100

= 550 − 100
