Unter allen Umständen ist die Umschlagszeit, wenn C mehr als einmal
umschlägt im Jahr ein ächter Bruch, da das Jahr Einheit ist für die
Berechnung der Umschlagsanzahl. Läuft C 3 × im Jahr um, so in 1 3  Jahr
einmal. Läuft C nicht einmal im Jahr um, so läuft es in mehr als einem
Jahr einmal um, u. seine Umschlagsperiode ein unächter Bruch, dessen
Zähler grösser als sein Nenner.

Die Umschlagszeit nur gleich 1, wenn C einmal im Jahr umschlägt.

Die Umschlagszeit ist sonst x n , wo x < 1, wenn C mehr als einmal im
Jahr umschlägt, > als 1 wenn es weniger als einmal im Jahr umschlägt. In
allen Fällen aber ist U = 1 (d. Jahr) x n , also die jährliche Umschlagsan-
zahl = n x = dem umgekehrten Bruch, der die Umschlagszeit anzeigt.

Nun wissen wir aber UC = K, wo K der Kostpreis = der Grösse des
umgeschlagnen Kapitals, i. e. = der Grösse des vorgeschoßnen Kapitals
multiplicirt mit der Anzahl seiner Umschläge.

Da aber U = n x , so UC = n x C.

∴ n x C = K, wenn die Umschlagszeit (die Zeit, worin C einmal um-
schlägt) = x n .

∴ Das vorgeschossne Kapital C × mit der umgekehrten Umschlagszeit
( x n ) = der Grösse des in einem Jahr umgeschlagnen Kapitals. Ist U = 1, so
die Umschlagszeit = 1 und das umgekehrte 1 1 auch = 1.

Ist x < n, also x n ein ächter Bruch, so ist n x ein unächter Bruch, daher n x >
1, daher n x C > C, also K, das umgeschlagne Kapital, grösser als C.

Ist x > n, also x n ein unächter Bruch, so ist n x ein ächter Bruch u. daher
n x C < C, also K, das umgeschlagne Kapital, < C.
