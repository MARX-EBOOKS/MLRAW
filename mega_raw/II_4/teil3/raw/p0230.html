Hier p constant u. π wechselt mit dem Umschlag, der nur verschiedner
Umschlagszeit des fixen Kapitals, also auch verschiedner Grösse seines
jährlichen Dechets geschuldet ist.

Schlägt das Kapital einmal im Jahr um, so stets π = p. Schlägt das
Kapital mehr als einmal im Jahr [um], so stets K = C + δ od. C = K – δ.
Und schlägt das Kapital weniger als einmal im Jahr um, so immer K = C
– δ od. C = K + δ. Aber δ hat hier einen andren Sinn als vorher.

Da die Umschlagsanzahl (ebenso seine Umschlagszeit) des cirkuliren-
den Kapitals gegeben ist, z. B. in unsrem Beispiel 4 + 3 5  mal, also auch
die Grösse des umgeschlagnen cirkulirenden Kapitals, hier = 460 (360cc +
100v), so ist immer eine bestimmte Umschlagszeit, hence jährlicher Dechet
des fixen Kapitals gegeben, deren Grösse dem einmaligen Umschlag des
gesammten Kapitalwerths im Jahr entspricht.

Und zwar ist diese Grösse des Dechet (entsprechend der Umschlagszeit
des fixen Kapitals) dann so bestimmt, daß die Summe f, der jährliche
Dechet des fixen Kapitals + des umgeschlagnen cirkulirenden Kapitals =
C, od. C – dem umgeschlagnen cirkulirenden Kapital = f ist. f + n ×
cirkulirendes Kapital = C = K, dem Kostpreis des jährlichen Produkts.

Wenn f + n × cirkulirendes Kapital = K   = C, so:

(f + δ) + n × cirkulirendes Kapital = K′ = C + δ. od. K – δ = C

(f – δ) + n × cirkulirendes Kapital = K′′ = C – δ. od. K + δ = C
